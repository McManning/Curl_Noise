\subsection{Detailed Description}
An object positioned in 3\-D space. 

This object can not be rendered by itself. It needs a child or children in form of meshes (\hyperlink{class_triangle_mesh}{Triangle\-Mesh} or \hyperlink{class_line_mesh}{Line\-Mesh}). All objects inheriting from \hyperlink{class_object3_d}{Object3\-D} can be added as child.
