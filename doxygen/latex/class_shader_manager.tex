\subsection{Detailed Description}
A class containing all shader program I\-Ds in a map. 

This class uses the singleton design pattern. This means that It is not possible to create instances of the class. There is only one instance that is reached from the static function \hyperlink{class_shader_manager_ad0c27625f06cc78bf8716485f5bce8e2}{instance()}.
