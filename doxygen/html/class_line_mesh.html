Detailed Description

This class extends AbstractMesh and renders lines.

Currently, this mesh can not be loaded from file.

Constructor & Destructor Documentation

LineMesh::LineMesh ( )

Creates a LineMesh without initializing vertex lists.

This constructor is used when creating basic LineMeshes such as grid-planes or axes. First construct the LineMesh, then call one of the init-functions (eg: initAxes, initGridPlane).
